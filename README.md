# 📊 E-commerce Sales Dashboard

## Project Overview

This project analyzes e-commerce sales data using Power BI to uncover business insights.

---

## Tools Used

- Power BI
- Excel
- Data Cleaning
- Data Visualization
- DAX

---

## Dashboard Features

- Total Sales
- Total Profit
- Total Orders
- Sales by Category
- Sales by Region
- Monthly Sales Trend
- Top Customers
- Top Products

---

## Business Insights

- Identified highest selling categories.
- Compared sales across regions.
- Found top customers.
- Tracked monthly sales trends.
- Analyzed profit performance.
